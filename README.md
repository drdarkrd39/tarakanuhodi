# 🪳 tarakanuhodi — Игровые проекты / Game Projects

Браузерные HTML5-игры для сайта [tarakanuhodi.kz](https://tarakanuhodi.kz) — компании по дезинфекции и борьбе с вредителями.

Browser-based HTML5 games for [tarakanuhodi.kz](https://tarakanuhodi.kz) — a pest control company.

---

## 🎮 Файлы / Files

### 🧪 `tetris_d.html` — Дезинфекционный тетрис
**RU:** Классический тетрис в тёмном стиле с насекомыми на блоках. Поддержка клавиатуры (стрелки, WASD, ЦЫФВ), сенсорных кнопок на мобильных, свайпов. Онлайн-таблица лидеров, система скидок по итогам игры.

**EN:** Classic Tetris with a dark pest-control aesthetic — insect emojis on every block. Supports keyboard (arrows, WASD, Russian layout), mobile touch buttons, and swipe gestures. Online leaderboard, discount system based on game results.

---

### 🐜 `survivor_easy.html` — Дезинфектор (облегчённая версия)
**RU:** Упрощённая версия основной игры с более редким спауном врагов и увеличенным бонусом за гемы. Содержит бестиарий с описанием всех врагов и боссов. Подходит для новых игроков.

**EN:** A reduced-difficulty version of the main game with slower enemy spawns and increased gem bonuses. Includes a bestiary with descriptions of all enemies and bosses. Suitable for new players.

---

### 🏆 `prize_dashboard.html` — Дашборд призёров
**RU:** Административная страница для просмотра списка лучших игроков, получивших скидки. Фильтры по номеру телефона, месту, дате, игре. Показывает скидку каждого игрока (30% / 15% / 5%). Автообновление каждые 30 секунд.

**EN:** Admin dashboard for viewing the list of prize-winning players. Filters by phone number, position, date, and game. Displays each player's discount (30% / 15% / 5%). Auto-refreshes every 30 seconds.

---

## ⚙️ Технологии / Tech Stack

- HTML5 Canvas, JavaScript (no frameworks)
- [Supabase](https://supabase.com) — облачная база данных / cloud database (PostgreSQL)
- Адаптивный дизайн, поддержка iOS / Responsive design, iOS support
